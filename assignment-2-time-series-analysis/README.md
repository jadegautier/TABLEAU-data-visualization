## Assignment 2 – Time Series Visualization in Tableau

**Course Context**  
Graduate-level Data Visualization course, Columbia University.

### Objective
Build a Tableau dashboard to visualize **year-to-date (YTD) price and trading volume trends** for the *Magnificent 7* U.S. technology companies.

### Companies Analyzed
- Alphabet (GOOG / GOOGL)  
- Amazon (AMZN)  
- Apple (AAPL)  
- Meta Platforms (META)  
- Microsoft (MSFT)  
- NVIDIA (NVDA)  
- Tesla (TSLA)

### Data Collection
- Financial data was collected using **Google Finance** formulas in Google Sheets:
  - Daily stock prices  
  - Daily trading volumes  
- Data was exported as CSV files and loaded into Tableau.

### Visualizations Built
- **Time series line charts** showing YTD stock price evolution  
- **Volume charts** illustrating daily trading activity  
- Combined dashboard for cross-company comparison  

### Key Insights
- Clear divergence in YTD performance across companies  
- Periods of high volatility correspond with spikes in trading volume  
- Side-by-side comparison improves trend detection across stocks  

### Tools
- Tableau  
- Google Sheets (Google Finance)  
- Time series analysis  

**Note:**  
Dashboards were built and submitted as a Tableau packaged workbook (`.twbx`).  
Visual documentation is provided in text form due to Tableau Public access limitations.
