# Tableau Data Visualization Projects

This repository contains a series of data visualization projects completed as part of a graduate-level course at **Columbia University**.  
The projects focus on applying visualization principles using **Tableau** to explore company data, financial time series, and geographic patterns.

Due to Tableau Public access limitations, interactive dashboards are not embedded here.  
Each project is documented with objectives, datasets, visual design choices, and key insights.

## Projects
- **Assignment 1:** Data & Chart Types  
- **Assignment 2:** Time Series Visualization (Magnificent 7 stocks)  
- **Assignment 3:** Geographic Mapping & Spatial Analysis  

## Tools & Skills
- Tableau  
- Data Visualization & Storytelling  
- Exploratory Data Analysis  
- Time Series Analysis  
- Geospatial Mapping
